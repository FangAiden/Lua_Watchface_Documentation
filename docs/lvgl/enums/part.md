# PART 枚举

`PART` 枚举定义了 LVGL 对象（Object）和控件的不同 **部件（Part）**。  
每个部件都可以单独设置样式（如颜色、边框、阴影等），用于实现更精细的外观控制。

---

## 枚举表

| 常量 | 值 | 说明 |
|------|----|------|
| **MAIN** | 0 | 主体部分（默认部分，通常是控件的背景或主要区域） |
| **SCROLLBAR** | 65536 | 滚动条部分 |
| **INDICATOR** | 131072 | 指示器部分（如滑条、进度条中的填充区域） |
| **KNOB** | 196608 | 控制柄部分（如滑块的圆形拖动点） |
| **ITEMS** | 327680 | 子项目部分（如列表或菜单项） |
| **SELECTED** | 262144 | 选中项部分（例如选中的按钮或列表项） |
| **CURSOR** | 393216 | 光标部分（如文本框中的光标） |
| **TICKS** | 458752 | 刻度线部分（如滑块或仪表刻度） |
| **CUSTOM_FIRST** | 524288 | 用户自定义部件起始值（用于扩展自定义样式） |
| **ANY** | 983040 | 匹配任意部件（用于通用样式匹配） |
