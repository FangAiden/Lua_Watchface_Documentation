# KEYBOARD_MODE 枚举

`KEYBOARD_MODE` 用于设置 **虚拟键盘 (`lv_keyboard`) 的输入模式**。  
不同模式下，键盘会显示不同的按键布局（如小写字母、大写字母、数字或特殊符号）。

---

## 枚举表

| 常量 | 值 | 说明 |
|------|----|------|
| **TEXT_LOWER** | 0 | 小写字母模式（默认） |
| **TEXT_UPPER** | 1 | 大写字母模式 |
| **SPECIAL** | 2 | 特殊符号模式（如 @、#、$、% 等） |
| **NUMBER** | 3 | 数字键盘模式（仅显示 0–9 及常用符号） |
| **USER_1** | 4 | 用户自定义键盘布局 1 |
| **USER_2** | 5 | 用户自定义键盘布局 2 |
| **USER_3** | 6 | 用户自定义键盘布局 3 |
| **USER_4** | 7 | 用户自定义键盘布局 4 |
