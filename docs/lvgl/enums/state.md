# STATE 枚举

`STATE` 枚举定义了 LVGL 对象的交互与视觉状态。  
这些状态可叠加使用（按位或 `|`），用于在不同交互情境下自动切换样式。  
例如，按钮在被按下、悬停或禁用时会显示不同的外观。

---

## 枚举表

| 常量 | 值 | 说明 |
|------|----|------|
| **DEFAULT** | 0 | 默认状态（无特殊状态时的基准样式） |
| **CHECKED** | 1 | 已选中状态（如开关开启、单选按钮被选中） |
| **FOCUSED** | 2 | 获得焦点状态（如通过键盘或编码器导航选中） |
| **FOCUS_KEY** | 4 | 键盘焦点状态（键盘/编码器控制下获得焦点） |
| **EDITED** | 8 | 编辑中状态（例如文本框正在输入） |
| **HOVERED** | 16 | 指针悬停状态（鼠标或触控笔悬停在对象上） |
| **PRESSED** | 32 | 按下状态（用户正在点击或触摸对象） |
| **SCROLLED** | 64 | 对象正在滚动时的状态 |
| **DISABLED** | 128 | 禁用状态（不可交互、通常为灰色外观） |
| **USER_1** | 4096 | 用户自定义状态 1（用于扩展特殊交互） |
| **USER_2** | 8192 | 用户自定义状态 2 |
| **USER_3** | 16384 | 用户自定义状态 3 |
| **USER_4** | 32768 | 用户自定义状态 4 |
| **ANY** | 65535 | 匹配任意状态（用于通用样式应用） |
