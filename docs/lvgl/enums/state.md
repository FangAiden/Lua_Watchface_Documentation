# STATE

| 常量 | 值 | 说明 |
|------|----|------|
| **DEFAULT** | 0 | 默认状态 |
| **CHECKED** | 1 | 已选中 |
| **FOCUSED** | 2 | 获得焦点 |
| **FOCUS_KEY** | 4 | 键盘焦点 |
| **EDITED** | 8 | 编辑中 |
| **HOVERED** | 16 | 指针悬停 |
| **PRESSED** | 32 | 按下状态 |
| **SCROLLED** | 64 | 滚动中 |
| **DISABLED** | 128 | 禁用状态 |
| **USER_1** | 4096 | 用户自定义状态 1 |
| **USER_2** | 8192 | 用户自定义状态 2 |
| **USER_3** | 16384 | 用户自定义状态 3 |
| **USER_4** | 32768 | 用户自定义状态 4 |
| **ANY** | 65535 | 匹配任意状态 |
