# FLEX_ALIGN

| 常量 | 值 | 说明 |
|------|----|------|
| **START** | 0 | 从起始位置对齐 |
| **END** | 1 | 从末尾位置对齐 |
| **CENTER** | 2 | 居中对齐 |
| **SPACE_EVENLY** | 3 | 间距均等（含两端） |
| **SPACE_AROUND** | 4 | 间距均等，首尾为一半 |
| **SPACE_BETWEEN** | 5 | 均匀分布，首尾紧贴边界 |
