# EVENT 枚举

LVGL 的事件（`lv_event_t`）用于对象交互、绘制、状态变化等场景。  
每个事件在特定时机触发，可通过 `lv_obj_add_event_cb()` 注册回调进行处理。

---

## 枚举表

| 常量 | 值 | 说明 |
|------|----|------|
| **ALL** | 0 | 匹配所有事件，用于监听器注册时捕获所有类型事件 |
| **PRESSED** | 1 | 对象被按下（首次触摸或点击时触发） |
| **PRESSING** | 2 | 按住对象时持续触发（长按过程中） |
| **PRESS_LOST** | 3 | 按下过程中指针离开对象区域，导致按压状态丢失 |
| **SHORT_CLICKED** | 4 | 快速点击（短按后释放）时触发 |
| **LONG_PRESSED** | 5 | 长按达到阈值（通常约 500 ms）时触发一次 |
| **LONG_PRESSED_REPEAT** | 6 | 长按期间持续重复触发 |
| **CLICKED** | 7 | 按下并释放完成点击时触发 |
| **RELEASED** | 8 | 按下后释放，无论指针是否仍在对象上 |
| **SCROLL_BEGIN** | 9 | 开始滚动时触发（如 `List`、`Page` 等可滚动对象） |
| **SCROLL_END** | 11 | 滚动停止时触发 |
| **SCROLL** | 12 | 滚动过程中持续触发 |
| **GESTURE** | 13 | 检测到手势（滑动、捏合等）时触发 |
| **KEY** | 14 | 对象接收到键盘或编码器输入事件 |
| **FOCUSED** | 16 | 对象获得焦点时触发 |
| **DEFOCUSED** | 17 | 对象失去焦点时触发 |
| **LEAVE** | 18 | 指针离开对象区域时触发 |
| **HIT_TEST** | 19 | 命中测试阶段触发，用于决定事件是否传递给对象 |
| **COVER_CHECK** | 21 | 检查对象是否完全覆盖在其他对象之上 |
| **REFR_EXT_DRAW_SIZE** | 22 | 计算对象额外绘制区域（如阴影、轮廓）时触发 |
| **DRAW_MAIN_BEGIN** | 23 | 绘制主区域开始前触发 |
| **DRAW_MAIN** | 24 | 绘制主区域时触发（主要绘图阶段） |
| **DRAW_MAIN_END** | 25 | 绘制主区域结束后触发 |
| **DRAW_POST_BEGIN** | 26 | 后绘制阶段开始（如叠加层）时触发 |
| **DRAW_POST** | 27 | 后绘制阶段进行中触发 |
| **DRAW_POST_END** | 28 | 后绘制阶段完成时触发 |
| **DRAW_PART_BEGIN** | 29 | 绘制部分（如样式或子区域）开始时触发 |
| **VALUE_CHANGED** | 30 | 对象值变化（如滑条、输入框、开关）时触发 |
| **INSERT** | 31 | 数据或文本插入（如 `TextArea` 输入）时触发 |
| **REFRESH** | 32 | 请求刷新对象外观时触发 |
| **READY** | 33 | 动作完成（如动画结束、输入完成）时触发 |
| **CANCEL** | 34 | 操作被取消（如输入中断或加载终止）时触发 |
| **DELETE** | 36 | 对象即将被删除时触发（可用于清理资源） |
| **CHILD_CHANGED** | 37 | 子对象属性（如大小、位置）变化时触发 |
| **CHILD_CREATED** | 38 | 创建新子对象时触发 |
| **CHILD_DELETED** | 39 | 删除子对象时触发 |
| **SCREEN_LOAD_START** | 41 | 屏幕开始加载时触发 |
| **SCREEN_LOADED** | 42 | 屏幕加载完成后触发 |
| **SCREEN_UNLOADED** | 43 | 屏幕卸载完成后触发 |
| **SIZE_CHANGED** | 44 | 对象大小发生变化时触发 |
| **STYLE_CHANGED** | 45 | 样式变化（如主题或 `style_set` 更新）时触发 |
| **LAYOUT_CHANGED** | 46 | 布局更新（如 Flex/Grid 调整）后触发 |
| **GET_SELF_SIZE** | 47 | 请求对象自行计算理想大小（auto-size）时触发 |
