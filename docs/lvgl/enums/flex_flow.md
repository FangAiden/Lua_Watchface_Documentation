# FLEX_FLOW

| 常量 | 值 | 说明 |
|------|----|------|
| **ROW** | 0 | 水平排列，从左到右 |
| **COLUMN** | 1 | 垂直排列，从上到下 |
| **ROW_WRAP** | 4 | 水平排列，自动换行 |
| **COLUMN_WRAP** | 5 | 垂直排列，自动换列 |
| **ROW_REVERSE** | 8 | 水平反转排列 |
| **COLUMN_REVERSE** | 9 | 垂直反转排列 |
| **ROW_WRAP_REVERSE** | 12 | 水平换行反转 |
| **COLUMN_WRAP_REVERSE** | 13 | 垂直换列反转 |
