# FLEX_FLOW 枚举

`FLEX_FLOW` 用于设置 **Flex 容器的主轴方向** 和 **是否换行**。  
它控制子对象的排列顺序（水平或垂直）以及在空间不足时的换行行为。

---

## 枚举表

| 常量 | 值 | 说明 |
|------|----|------|
| **ROW** | 0 | 子对象水平排列，从左到右（默认） |
| **COLUMN** | 1 | 子对象垂直排列，从上到下 |
| **ROW_WRAP** | 4 | 水平排列，空间不足时自动换行到下一行 |
| **COLUMN_WRAP** | 5 | 垂直排列，空间不足时自动换列 |
| **ROW_REVERSE** | 8 | 水平排列，顺序反转（从右到左） |
| **COLUMN_REVERSE** | 9 | 垂直排列，顺序反转（从下到上） |
| **ROW_WRAP_REVERSE** | 12 | 水平排列并换行，顺序反转（从右到左） |
| **COLUMN_WRAP_REVERSE** | 13 | 垂直排列并换列，顺序反转（从下到上） |
