# SCR_LOAD_ANIM 枚举

`SCR_LOAD_ANIM` 定义了 **屏幕切换动画类型**，用于 `lv_scr_load_anim()` 函数。  
通过选择不同的动画方式，可以在界面切换时实现平滑的过渡效果。

---

## 枚举表

| 常量 | 值 | 说明 |
|------|----|------|
| **NONE** | 0 | 无动画，直接切换到新屏幕 |
| **OVER_LEFT** | 1 | 新屏幕从左侧覆盖旧屏幕 |
| **OVER_RIGHT** | 2 | 新屏幕从右侧覆盖旧屏幕 |
| **OVER_TOP** | 3 | 新屏幕从上方覆盖旧屏幕 |
| **OVER_BOTTOM** | 4 | 新屏幕从下方覆盖旧屏幕 |
| **MOVE_LEFT** | 5 | 旧屏幕左移退出，新屏幕从右侧进入 |
| **MOVE_RIGHT** | 6 | 旧屏幕右移退出，新屏幕从左侧进入 |
| **MOVE_TOP** | 7 | 旧屏幕上移退出，新屏幕从下方进入 |
| **MOVE_BOTTOM** | 8 | 旧屏幕下移退出，新屏幕从上方进入 |
| **FADE_ON** / **FADE_IN** | 9 | 渐入动画，新屏幕淡入显示 |
| **FADE_OUT** | 10 | 渐出动画，旧屏幕淡出隐藏 |
| **OUT_LEFT** | 11 | 新屏幕固定，旧屏幕向左滑出 |
| **OUT_RIGHT** | 12 | 新屏幕固定，旧屏幕向右滑出 |
| **OUT_TOP** | 13 | 新屏幕固定，旧屏幕向上滑出 |
| **OUT_BOTTOM** | 14 | 新屏幕固定，旧屏幕向下滑出 |
