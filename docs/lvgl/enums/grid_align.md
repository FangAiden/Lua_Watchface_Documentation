|------|----|------|
| **START** | 0 | 靠起始边对齐 |
| **CENTER** | 1 | 居中对齐 |
| **END** | 2 | 靠末端对齐 |
| **STRETCH** | 3 | 拉伸填充 |
| **SPACE_EVENLY** | 4 | 间距均等（含边界） |
| **SPACE_AROUND** | 5 | 间距均等，首尾为一半 |
| **SPACE_BETWEEN** | 6 | 首尾紧贴，间距均分 |
