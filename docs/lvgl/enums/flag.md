# FLAG 枚举

对象标志（`lv_obj_flag_t`）用于控制对象的行为特性和交互方式。  
这些标志可通过 `lv_obj_add_flag()`、`lv_obj_clear_flag()`、`lv_obj_has_flag()` 等函数进行设置或查询。

---

## 枚举表

| 常量 | 值 | 说明 |
|------|----|------|
| **HIDDEN** | 1 | 隐藏对象（对象不可见但仍存在） |
| **CLICKABLE** | 2 | 对象可被点击（产生点击事件） |
| **CLICK_FOCUSABLE** | 4 | 点击对象时可获得焦点 |
| **CHECKABLE** | 8 | 对象可切换选中状态（如复选框） |
| **SCROLLABLE** | 16 | 对象内容可滚动 |
| **SCROLL_ELASTIC** | 32 | 启用滚动回弹效果 |
| **SCROLL_MOMENTUM** | 64 | 启用滚动惯性（滑动时自动减速） |
| **SCROLL_ONE** | 128 | 一次只能滚动一个方向（防止斜向滚动） |
| **SCROLL_CHAIN_HOR** | 256 | 允许横向滚动链（父子滚动同步） |
| **SCROLL_CHAIN_VER** | 512 | 允许纵向滚动链 |
| **SCROLL_ON_FOCUS** | 1024 | 当对象获得焦点时自动滚动至可见区域 |
| **SCROLL_WITH_ARROW** | 2048 | 允许通过方向键进行滚动 |
| **SNAPPABLE** | 4096 | 启用对齐吸附功能（滚动或拖动时自动对齐） |
| **PRESS_LOCK** | 8192 | 按下后锁定状态，不自动释放 |
| **EVENT_BUBBLE** | 16384 | 事件冒泡：事件可传递至父级对象 |
| **GESTURE_BUBBLE** | 32768 | 手势冒泡：手势事件可向上传递 |
| **OVERFLOW_VISIBLE** | 1048576 | 内容超出对象边界时仍可显示（不裁剪） |
| **FLOATING** | 262144 | 对象浮动于布局系统之外（不参与布局） |
| **IGNORE_LAYOUT** | 131072 | 忽略父容器布局影响 |
| **LAYOUT_1** | 8388608 | 用户自定义布局标志 1（可用于特殊布局控制） |
| **LAYOUT_2** | 16777216 | 用户自定义布局标志 2 |
| **WIDGET_1** | 33554432 | 用户自定义控件标志 1 |
| **WIDGET_2** | 67108864 | 用户自定义控件标志 2 |
| **USER_1** | 134217728 | 用户自定义标志 1 |
| **USER_2** | 268435456 | 用户自定义标志 2 |
| **USER_3** | 536870912 | 用户自定义标志 3 |
| **USER_4** | 1073741824 | 用户自定义标志 4 |
