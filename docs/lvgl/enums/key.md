# KEY 枚举

`KEY` 枚举用于定义 **键盘输入** 或 **旋钮编码器（encoder）输入** 的按键代码。  
这些按键常量可在事件回调中通过 `lv_event_get_key()` 获取，用于判断用户输入。

---

## 枚举表

| 常量 | 值 | 说明 |
|------|----|------|
| **HOME** | 2 | 跳转到起始位置（常用于文本或列表） |
| **END** | 3 | 跳转到末尾位置 |
| **BACKSPACE** | 8 | 删除光标前的字符 |
| **ENTER** | 10 | 确认输入或执行操作（如点击） |
| **NEXT** | 9 | 跳转到下一个控件（Tab 功能） |
| **PREV** | 11 | 跳转到上一个控件 |
| **ESC** | 27 | 取消、退出或关闭当前操作 |
| **DEL** | 127 | 删除光标后的字符 |
| **UP** | 17 | 向上导航或移动焦点 |
| **DOWN** | 18 | 向下导航或移动焦点 |
| **RIGHT** | 19 | 向右导航或移动焦点 |
| **LEFT** | 20 | 向左导航或移动焦点 |
