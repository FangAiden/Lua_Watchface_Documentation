# KEY

| 常量 | 值 | 说明 |
|------|----|------|
| **HOME** | 2 | 跳转到起始位置 |
| **END** | 3 | 跳转到末尾位置 |
| **BACKSPACE** | 8 | 删除光标前字符 |
| **NEXT** | 9 | 下一个控件 |
| **ENTER** | 10 | 确认/执行 |
| **PREV** | 11 | 上一个控件 |
| **UP** | 17 | 向上 |
| **DOWN** | 18 | 向下 |
| **RIGHT** | 19 | 向右 |
| **LEFT** | 20 | 向左 |
| **ESC** | 27 | 取消/退出 |
| **DEL** | 127 | 删除光标后字符 |
