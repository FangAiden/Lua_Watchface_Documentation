# DIR 枚举

`DIR`（Direction）用于描述方向标志，可用于滚动、布局、拖动、动画等场景。  
这些值通常可以组合使用（按位或 `|`）。

---

## 枚举表

| 常量 | 值 | 说明 |
|------|----|------|
| **NONE** | 0 | 无方向（禁用或不限定方向） |
| **LEFT** | 1 | 向左方向 |
| **RIGHT** | 2 | 向右方向 |
| **HOR** | 3 | 水平方向（等同于 `LEFT \| RIGHT`） |
| **TOP** | 4 | 向上方向 |
| **BOTTOM** | 8 | 向下方向 |
| **VER** | 12 | 垂直方向（等同于 `TOP \| BOTTOM`） |
| **ALL** | 15 | 所有方向（等同于 `HOR \| VER`） |
