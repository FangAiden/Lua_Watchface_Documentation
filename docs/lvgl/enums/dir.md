# DIR

| 常量 | 值 | 说明 |
|------|----|------|
| **NONE** | 0 | 无方向 |
| **LEFT** | 1 | 向左方向 |
| **RIGHT** | 2 | 向右方向 |
| **HOR** | 3 | 水平方向（LEFT \| RIGHT） |
| **TOP** | 4 | 向上方向 |
| **BOTTOM** | 8 | 向下方向 |
| **VER** | 12 | 垂直方向（TOP \| BOTTOM） |
| **ALL** | 15 | 所有方向（HOR \| VER） |
